# Privacy Policy

aict is a read-only CLI tool that outputs structured data (XML/JSON) for AI agents.

## Data Collection

aict does not collect, store, or transmit any user data. It is a read-only tool that:
- Reads files and directories you explicitly specify
- Outputs structured data to stdout
- Does not make network requests (except for local MIME type detection)
- Does not log or store any information

## Usage

When you run aict with any command:
- The tool reads the files/directories you specify
- It processes the content and outputs structured data
- No data is sent anywhere or stored persistently

## Third Parties

aict does not share any data with third parties. It is a standalone CLI tool with no telemetry, analytics, or external dependencies.

## Changes

If this policy changes, it will be updated in this repository.

## Contact
